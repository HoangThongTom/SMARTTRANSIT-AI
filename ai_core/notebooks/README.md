# 📓 Notebooks — SmartTransit AI EDA

Thư mục này chứa các Jupyter Notebook để phân tích dữ liệu giao thông TP.HCM.

## Hướng dẫn chạy

```bash
pip install jupyter
cd smarttransit-ai
jupyter notebook ai_core/notebooks/
```

## Danh sách notebooks đề xuất

| File | Nội dung |
|------|----------|
| `01_EDA_traffic.ipynb` | Khám phá dữ liệu giao thông thô |
| `02_congestion_analysis.ipynb` | Phân tích mẫu kẹt xe theo giờ/ngày |
| `03_feature_importance.ipynb` | Xem tầm quan trọng features |
| `04_model_comparison.ipynb` | So sánh RF vs XGBoost vs LogReg |

> 💡 Tạo notebook mới bằng cách copy file `.ipynb.example` trong thư mục này.
